# 道具赛MVP分计算方式
来源: [组队道具mvp分计算方式](https://tieba.baidu.com/p/6455555391)     [个人道具赛mvp分计算方式](https://tieba.baidu.com/p/6457567536)  
  
x = 攻击+干扰+防御  
n = 名次  

## 组队道具
 
胜利: MVP分 = (x+5)\*(1.8-0.1n)  
失败: MVP分 = x\*(1.8-0.1n)  
  
## 个人道具 
完成: MVP分 = (x+6)\*(1.8-0.1n)  
未完成: MVP分 = (x+1)\*(1.8-0.1n)  
  
注: 显示时小数向下取整，同分时比小数部分，小数部分相同时比名次
